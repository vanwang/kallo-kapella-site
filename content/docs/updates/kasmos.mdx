# KĀSMOS更新说明


## Version 2.3.8

发布于 **2024-3-2**

💄 微信用户信息获取优化

💄 UI/UX更新：针对用户反馈进行了界面和用户体验的大幅度优化。

💄 优化管理员绑定操作


## Version 2.3.1

发布于 **2024-1-21**

✨ 支持多商户身份切换

💄 优化权限控制

💄 修复了一些用户反馈的小bug，提升系统稳定性。

## Version 2.2.3

发布于 **2023-7-28**

✨ 支持多设备控制

✨ 支持配方模糊查询

✨ 故障诊断及显示功能，用户可通过APP查看设备运行状态。

💄 性能改进：减少了系统体积，以及应用启动和响应时间。



## Version 2.1.0

发布于 **2023-6-25**

✨ 增加商品列表、商品详情等商城功能

## Version 2.0.0

发布于 **2023-6-21**

✨ 用户界面：全新设计的用户界面，支持网络配方下发，提升了操作的直观性和流畅性。

🐞 修复问题：解决了在某些情况下配方同步失败的bug。

## Version 1.2.5

发布于 **2023-5-29**

💄 增加二次注水时间

💄 优化启动时间


## Version 1.1.1

发布于 **2023-4-4**


💄 调整上海门店配方


## Version 1.1.0

发布于 **2022-10-18**

✨ 集成配方存储功能

💄 可控制咖啡机制作指令


## Version 1.0.0

发布于 **2022-9-23**

✨ KĀSMOS单机版

支持蓝牙分包发送，可操作大指令集。

💄 优化系统启动时间。

🐞 解决了管理员端和商家端的权限限制问题。

🐞 解决了管理员端和商家端的页面访问偶尔会出现错误的问题。
