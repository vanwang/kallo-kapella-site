# KĀPELLA更新说明

## Version 2.3

发布于 **2024-2-6**

✨ 增加了设备状态的接口包括等待，制作中，完成等状态。

✨ 增加低温水箱温度调整和读取接口。


## Version 2.2

发布于 **2024-1-5**

💄 升级了研磨度光感挡片，防止由于研磨器震动导致档片不准。

💄 调整了柜门的封边焊接形式。

💄 调整了咖啡机落豆口的管道配置，管道可以从设备内部与台面落豆口连接。


## Version 2.1

发布于 **2023-10-1**

💄 升级了自动研磨度调节模块，改用红外光感识别研磨度。

💄 升级了咖啡注水出水方式，升级热水泵让萃取咖啡水量更精准。


## Version 2.0

发布于 **2023-5-1**

✨ 双锅炉设计，低温锅炉预加热高温锅炉加热到冲泡温度，保证设备的连续萃取能力。

✨ 蓝牙模块加入，设备连接系统实现无线操作。

✨ 自动研磨调节，产品制作随配方调整研磨度，保障不同咖啡豆都能有最佳萃取。


## Version 1.2

发布于 **2022-7-2**

💄 制作流程优化，标准产品制作由2分30秒减少至1分50秒。



## Version 1.0

发布于 **2022-5-1**

✨ 半自动研磨调节，3档研磨度。

✨ 咖啡机上水模式增加直饮水上水。
